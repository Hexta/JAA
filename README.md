JAA
===

Just Another Archiver

===

Archivers study app.

Include support compression methods: 
  * BWT
  * MTF
  * RLE
  * Huffman.
Allow use its in any combinations.

===

Supported platforms:
  * Windows XP-7(Tested)
   - MinGW x86, amd64
   - MSVC 2013
  * GNU/Linux (GCC 4.7+, Qt 5.0) x86, amd64



Installation

<code>cmake -DCMAKE_BUILD_TYPE=Release . && make && make install</code>
